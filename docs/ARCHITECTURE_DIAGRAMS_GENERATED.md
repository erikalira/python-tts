# Generated Architecture Diagrams

This guide points to the fully automatic architecture diagrams generated by `pyreverse`.

Use these when you want a fresh, low-maintenance snapshot of the codebase without hand-curating the result.

## Generated set

- [generated/README.md](diagrams/generated/README.md)
- [generated/shared.md](diagrams/generated/shared.md)
- [generated/bot-runtime.md](diagrams/generated/bot-runtime.md)
- [generated/desktop-runtime.md](diagrams/generated/desktop-runtime.md)

## Regeneration

```powershell
python scripts/utils/generate_architecture_diagrams.py
```

## Notes

- the generated diagrams are broader and noisier than the curated architecture diagrams
- they are useful for drift detection and quick inspection
- the curated diagrams in [ARCHITECTURE_DIAGRAMS.md](ARCHITECTURE_DIAGRAMS.md) remain the better onboarding view
- only the rendered `.md` files are kept; intermediate `.mmd` output is discarded by the generator
