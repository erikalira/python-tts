# Generated Architecture Diagrams

Use this page as the entrypoint for the automatically generated diagram set.

For the generated index and file list, go to:

- [docs/architecture/diagrams/generated/README.md](diagrams/generated/README.md)

For curated, human-maintained diagrams that explain the intended architecture,
prefer:

- [ARCHITECTURE_DIAGRAMS.md](ARCHITECTURE_DIAGRAMS.md)

## When to use generated diagrams

Use the generated set when you want:

- a quick structural snapshot of the codebase
- drift detection after architectural changes
- a broad import/class view without manual curation

Use the curated diagrams when you want onboarding, explanation, and intended
boundaries.
