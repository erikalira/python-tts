# 🎯 TTS Hotkey Premium - A Solução Definitiva

## 🏆 Configuração Profissional Sem Hardcode

**Entry point**: `app.py` + `scripts/build/build_clean_architecture.ps1`

### ✨ **A Solução Perfeita:**

- 🎯 **Classe Config centralizada** - tudo em um local elegante
- 🔧 **Zero hardcode** - personalize qualquer aspecto
- 🎨 **Interface profissional** - logs coloridos e informativos
- ⚡ **Configurações avançadas** - triggers, timeouts, perfis
- 🎮 **Perfis especializados** - Gaming, Office, Streaming
- 🚀 **Build inteligente** - analisa sua configuração automaticamente

### 🚀 Como usar:

```powershell
# 1. Ajuste a configuração do app se necessário
code app.py

# 2. Execute o build inteligente
./scripts/build/build_clean_architecture.ps1

# 3. Um único arquivo pronto!
dist/tts_hotkey_clean.exe
```

### 🎨 Painel principal do Desktop App

Ao abrir o executável, o usuário vê um painel principal que:

- permanece aberto como janela principal do app
- permite configurar Discord, TTS, hotkeys e preferências de interface
- oferece `Testar conexão` sob demanda, sem polling contínuo
- oferece `Enviar teste de voz` manual com mensagem curta
- exibe atividade e logs úteis sem depender do terminal

Use a configuração persistida do Desktop App em `src/standalone/config/standalone_config.py`
e a interface gráfica do próprio app para ajustar Discord, TTS e hotkeys.

### 🎮 **Perfis Especializados Incluídos:**

- **🎮 Gaming**: Triggers `[texto]`, sem notificações, resposta rápida
- **💼 Office**: Configuração profissional, logs limpos, mais tolerante
- **🎥 Streaming**: Integração com OBS, virtual cable, otimizado
